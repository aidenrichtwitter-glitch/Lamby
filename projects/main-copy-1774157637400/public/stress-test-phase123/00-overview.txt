HOW THE BRIDGE WORKS:
  1. User clicks "Copy Context" in Lamby → prompt copied to clipboard
  2. User pastes prompt into Grok on grok.com
  3. Grok reads the prompt, sees bridge endpoint URLs
  4. Grok uses its browse_page tool (GET-only) to call bridge endpoints
  5. Bridge relay forwards requests to user's desktop via WebSocket
  6. Changes happen on the user's local project files
  THIS APP DOES NOT CALL GROK'S API FOR BRIDGE CHANGES.
  GROK DOES ALL THE WORK ITSELF THROUGH BROWSE_PAGE.
