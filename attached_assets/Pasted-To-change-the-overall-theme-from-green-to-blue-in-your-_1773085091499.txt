To change the overall theme from green to blue in your Guardian AI / λ Recursive app, the main place to update is the CSS variables in `src/index.css`.

Here's the updated file with a clean blue-oriented palette (cool dark background + blue accents that still feel "cyber / terminal" but shifted away from green):

```css
// file: src/index.css
/* Changed from terminal-green to terminal-blue theme
   Dark void background with blue glow / accents */

:root {
  --background: 220 20% 8%;       /* Slightly deeper dark blue-gray */
  --foreground: 210 40% 85%;      /* Soft blue-white for text */
  --primary:    210 80% 55%;      /* Main interactive blue */
  --terminal-green: 210 70% 60%;  /* Renamed → now the primary blue (used where old green was) */
  --terminal-cyan:  200 80% 65%;  /* Brighter cyan-blue for secondary highlights */
  --muted:      220 15% 45%;      /* Subtle muted blue-gray */
  --border:     220 30% 25%;      /* Darker blue borders */
}

/* If you have any hardcoded hsl/hsla colors elsewhere in components,
   look for values around hsl(140, 70%, 45%) and replace with something like
   hsl(210, 80%, 55%) or var(--primary) */

body {
  background-color: hsl(var(--background));
  color: hsl(var(--foreground));
}
```
